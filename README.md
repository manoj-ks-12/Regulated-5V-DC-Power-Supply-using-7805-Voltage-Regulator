Designed and simulated a regulated DC power supply circuit that converts AC input to a stable 5V DC output using a step-down transformer, bridge rectifier (1N4007 diodes), filtering capacitors, and a 7805 voltage regulator IC. The project includes PCB design implementation and circuit simulation analysis to verify output stability and ripple reduction. Oscilloscope results were used to analyze waveform behavior at different stages of conversion. The design ensures reliable and noise-free DC output suitable for electronic devices and embedded systems.

Tools Used: Proteus Design Suite / Circuit Simulation Software, PCB Design Tools
Key Skills: Analog Electronics, Power Supply Design, Circuit Simulation, PCB Design, Signal Analysis
